Ninja Dodge (working title)
===

CONTROLS

- **W** *or* **Up**: move up  
- **A** *or* **Left**: move left  
- **S** *or* **Down**: move down  
- **D** *or* **Right**: move right  
- **Left Mouse Button**: shoot projectile  
- **F**: teleport player to cursor  
- **H**: HUD on/off

Just run ***index.html*** to start the game.
